# TurystyczneTriCity

# Requirements:
- node >=6.0
- npm >=3.0
- yarn >=0.2

# Instruction:
1. git clone
2. Include your own Google Maps and Google Firebase API keys in sercret.js file
3. yarn
4. npm run buildvendor
5. npm start / yarn start

# Includes:
- React
- Redux
- SASS
